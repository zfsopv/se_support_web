# 微服务器SE7 32-BP1产品规格书-v1.0.3

## 微服务器 SE7

### 超高性能

- 支持INT8、FP16/BF16、FP32
- 支持32路1080P@25fps高清视频硬件解码能力
- 支持12路1080P@25fps高清视频硬件编码能力

### 灵活部署

- 宽温，IP40防护等级
- 无风扇设计，结合外壳被动散热
- Docker容器化，Kubernetes扩展管理

### 接口丰富

- 支持USB3.0 *2 / MicroSD *1 / HDMI *1 / RS-232 *1 / RS-485 *1 / Audio *1 / TYPE C*1 / Sim*1 / 自定义I/O *2
- 支持2.5寸或3.5寸SATA（外接硬盘仓）
- 支持M.2 SATA3.0 SSD（主机内部，可选尺寸2242/2260/2280）
- 支持4G、5G模块扩展

微服务器SE7 32-BP1系列搭载算丰第四代智能视觉深度学习处理器BM1684X，具备高性能、低功耗、环境适应性强等特点，通过搭配多样化深度学习算法，实现人脸识别、视频结构化、行为分析、状态监测等应用，为智慧城市、智慧交通、智慧能源、智慧金融、智慧电信、智慧工业进行数字化赋能。
