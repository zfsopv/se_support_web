## 1 安全

### 1.1 通用安全

#### 事项说明

在操作、安装本设备时，请遵循设备标识以及手册中说明的通用安全相关注意事项。

关于手册中提及的 "危险"、"警告"、"注意" 事项，只是对安全事项的补充，并不代表所有安全事项。

#### 安装要求

操作、安装、维护本设备的人员必须了解所有安全注意事项，并掌握正确的操作方法后才可进行操作。关于安装要求做以下几点说明：

- 设备操作、安装、维护必须由经过培训的人员完成。
- 设备检修必须由本公司授权人员完成。
- 设备部件更换必须由本公司授权人员完成。
- 设备因安装导致的设备故障、异常需由操作人员及时与我司联系。

#### 接地要求

- 设备安装前需要先完成接地操作，设备拆除时最后拆除地线。
- 严禁在设备未接地或接地不良的情况下对设备进行任何操作。
- 操作设备前请进行接地检查，确保设备接地良好。

#### 人身安全

- 设备安装、操作前，需要身穿防静电服，佩戴防静电手套或手环，摘除手部导电饰品，避免引起短路被电击损伤。
- 严禁在危险环境下操作、安装设备，如雷雨天气、易燃环境等。

#### 设备安全

- 保证设备在任何一种安装方式下的牢固可靠，如桌放、挂墙或机架安装。
- 对设备进行开机操作前确保接地良好，关机操作完成后再拆除地线。
- 维护窗操作需要佩戴防静电手套，拆卸或安装维护窗盖板请使用螺丝刀。
- 严禁遮挡、堵塞设备通风面板。
- 严禁拔除设备密封胶塞，如天线胶塞等。

### 1.2 电气安全

#### 恶劣环境

- 严禁在雷雨天气下对设备进行交流电操作，否则会有致命危险。
- 严禁在雷雨天气下对设备进行室外设备箱体、抱杆等作业，否则会有致命危险。
- 严禁在高压电源供电情况下，不规范地进行操作，否则会有致命危险。

#### 设备电源

- 严禁在设备上电状态下，进行带电拆除电源线操作，避免产生电火花导致身体损伤。
- 进行拆除设备电源线操作前，必须优先关闭电源使设备处于已关机状态下。
- 进行设备电源线连接设备操作前，必须优先确认电源线正负极标识正确再进行连接。
- 确保设备电源线与设备相连后，连接位置牢固不松动。

#### 静电

- 为防止人体摩擦、移动产生的静电损坏设备，对设备操作前，先进行静电放电操作。
- 触摸、手持设备前，必须佩戴防静电手环，并将手环的接地端子接地。

## 2 产品简介

### 2.1 产品概述

微服务器 SE7 是一款高性能、低功耗边缘计算产品。搭载国产智能视觉深度学习处理器 BM1684X，可同时处理 32 路 1080P 高清视频，支持 32 路高清视频硬件解码与 12 路编码。作为一款支持宽温、多接口的边缘计算设备，SE7 可灵活应对智能安防、智慧交通、智慧园区、智慧零售等多种场景，搭配多样化算法，实现人脸布控、视频结构化分析、商品识别等应用，为传统行业进行数字化赋能。

### 2.2 产品特点

#### 超高性能计算与编解码能力

- 支持 INT8/FP16/BF16/FP32
- 支持高达 32 路 H.264/H.265 1080P@25FPS 视频硬件解码
- 支持高达 12 路 H.264/H.265 1080P@25FPS 视频硬件编码

#### 接口丰富、部署灵活

- 支持丰富接口：1000M 以太网口、USB、HDMI、RS-485、RS-232、自定义 I/O 等
- 支持 -20°C 至 +70°C 工作环境（SE7 32-BP1-11）
- 支持 IP40 防护等级
- 支持 2.5 寸或 3.5 寸的 SATA 硬盘存储拓展（外接硬盘仓）
- 支持 M.2 SATA3.0 SSD（主机内部，可选尺寸 2242/2260/2280）
- 支持 4G、5G 模块扩展

#### 高可靠性，加密保护

- 高容量 eMMC 可开发支持主备分区
- 支持异常类故障告警与保护处理机制
- 支持可编程加密处理器用于隐私信息保护

#### 易用工具链，开发灵活

- 一站式深度学习开发工具包 Sophon SDK
- 支持 Caffe/DarkNet/TensorFlow/PyTorch/MXNet/ONNX/PaddlePaddle 等主流深度学习框架
- 支持分类、检测主流网络模型，支持自定义算子开发
- 支持 Docker 容器化，算法应用快速部署

## 5 系统配置

### 5.1 系统概述

- SE7 预装 Ubuntu 20.04，用户可以通过 SSH 登录或者串口登录，也可以通过 LiteOS 登陆。
- SE7 提供的 2 个网口，其中 WAN 口(eth0)：默认为 DHCP 动态获取 IP 由连接的路由分配。LAN 口(eth1)：默认为静态 IP，IP 地址为 192.168.150.1。
- SE7 默认登陆用户名/密码为 linaro/linaro 或 admin/admin。

### 5.2 系统登录

#### 5.2.1 SSH 登录

**说明**

首次登陆一般通过 LAN 口 IP 登陆，查看 WAN 口 IP 后可通过 WAN 口 IP 登陆。

**操作步骤**

- 步骤 1 准备两根网线，分别插入盒子 LAN 口和 WAN 口，其中 LAN 口网线接入电脑，WAN 口网线接入局域网内的交换机或者路由器。
- 步骤 2 配置电脑 IP 为 192.168.150.2，子网掩码为 255.255.255.0。
- 步骤 3 打开命令提示符，使用 ssh 登陆：ssh linaro@192.168.150.1，输入密码：linaro。
- 步骤 4 输入 ifconfig 命令查看 WAN 口 IP(eth0)，后续可以直接通过 WAN 口 IP 登陆。
- 结束

#### 5.2.2 串口登录

**说明**

用户可以自行选择串口终端登录工具，默认串口通讯参数为：115200-N-8-1。

**操作步骤**

- 步骤 1 使用 USB 转 Type-C 数据线连接 PC 机的 USB 口和 SE7 的 Type-C 调试口。
- 步骤 2 安装 PL2303_Prolific_DriverInstaller_v110 和 CH340SER 串口驱动(下载地址: https://pan.baidu.com/s/1sUvau1mVyoIm5Jn-kAKCeg)，安装后在设备管理器中就有串口显示。
- 步骤 3 配置 COM 口属性，115200-8-N-1。
- 步骤 4 下载 MobaXterm 软件，选择连接 COM，设置波特率: 115200。
- 步骤 5 输入账号密码进行登录。
- 结束

#### 5.2.3 Web 登录

SE7 Ubuntu 版本预装 LiteOS 后台管理系统，建议用户通过谷歌 Chrome 浏览器登录 LiteOS 系统，并进行系统配置。具体操作可参考《SOPHLITEOS 用户使用手册》。

#### 5.3.1 WAN IP 获取

**说明**

WAN IP 地址获取有多种方式，可以通过串口或 LAN 口 SSH 登录到系统终端后使用 ifconfig 命令查看（参考 5.2 章节），也可以通过 LiteOS 系统 web 界面查看、HDMI 界面查看和 IP Scanner 获取。本章节主要说明 HDMI 界面查看方式。

- 结束

#### 5.3.2 WAN IP 修改

**说明**

WAN IP 地址修改有多种方式，即 LiteOS 系统 web 页面修改、HDMI 界面修改；此外，您也可以通过串口或 SSH 登录到系统终端后，使用操作系统级别的通用命令进行修改。

- LAN IP 为 LiteOS 系统强制的静态 IP, LiteOS 系统修改 IP 的方式无法修改 LAN 口 IP。

- WAN IP 和 LAN IP 不能处于同一网段，必要时可 SSH 登录到系统终端并使用操作系统通用命令修改 LAN 口 IP，但建议 LAN IP 保持设置为静态 IP，以方便运维管理。

### 6.4 GPIO使用

如4.1.2节中描述，SE7提供了2对OUT/IN的GPIO端口，其中：

OUT1=1684x STM32PA2、OUT2=1684x STM32PA3固定为输出，IN1=GPIO7、IN2=GPIO5固定为输入。为了方便使用，预装软件提供了如下接口：

```
# 拉高OUT1
sudo i2cset -y -f 1 0x17 0x30 0山区

# 拉低OUT1
sudo i2cset -y -f 1 0x17 0x30 0xff

# 拉高OUT2
sudo i2cset -y -f 1 0x17 0x30 0xfd
```
