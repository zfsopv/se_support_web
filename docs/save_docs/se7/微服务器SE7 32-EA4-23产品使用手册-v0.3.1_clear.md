## 1 安全

### 1.1 通用安全

#### 事项说明

在操作、安装本设备时，请遵循设备标识以及手册中说明的通用安全相关注意事项。

关于手册中提及的 "危险" 、"警告" 、"注意" 事项，只是对安全事项的补充，并不代表所有安全事项。

#### 安装要求

操作、安装、维护本设备的人员必须了解所有安全注意事项，并掌握正确的操作方法后才可进行操作。关于安装要求做以下几点说明：

- 设备操作、安装、维护必须由经过培训的人员完成。
- 设备检修必须由本公司授权人员完成。
- 设备部件更换必须由本公司授权人员完成。
- 设备因安装导致的设备故障、异常需由操作人员及时与我司联系。

#### 人身安全

- 设备安装、操作前，需要身穿防静电服，佩戴防静电手套或手环，摘除手部导电饰品，避免引起短路被电击损伤。
- 严禁在危险环境下操作、安装设备，如雷雨天气、易燃环境等。

#### 设备安全

- 保证设备在任何一种安装方式下的牢固可靠，如桌放、挂墙或机架安装。
- 严禁遮挡、堵塞设备表面散热齿和散热风扇。
- 严禁拔除设备密封胶塞，如天线胶塞等。

### 1.2 电气安全

#### 恶劣环境

严禁在雷雨天气下对设备进行交流电操作，否则会有致命危险。

严禁在雷雨天气下对设备进行室外设备箱体、抱杆等作业，否则会有致命危险。

严禁在高压电源供电情况下，不规范地进行操作，否则会有致命危险。

#### ⚠️ 设备电源

严禁在设备上电状态下，进行带电拆除电源线操作，避免产生电火花导致身体损伤。

进行拆除设备电源线操作前，必须优先关闭电源使设备处于已关机状态下。

- 确保设备电源线与设备相连后，连接位置牢固不松动。

#### ⚠️静电

为防止人体摩擦、移动产生的静电损坏设备，对设备操作前，先进行静电放电操作。

- 触摸、手持设备前，必须佩戴防静电手环，并将手环的接地端子接地。

## 2 产品简介

### 2.1 产品概述

微服务器 SE7 32-EA4 是一款高性能、低功耗边缘计算产品。搭载 BM1684X 处理器，可同时处理 32 路高清视频，支持 32 路 1080P 高清视频硬件解码与 12 路编码。作为一款支持宽温、多接口的边缘计算设备，SE7 可应用于智能安防、智慧交通、智慧园区、安全生产等多种场景。

### 2.2 产品特点

#### 超高性能

- 支持 INT8/FP16/BF16/FP32;
- 支持高达 32 路 H.264/H.265 1080P@25FPS 视频硬件解码；
- 支持高达 12 路 H.264/H.265 1080P@25FPS 视频硬件编码。

#### 接口丰富、部署灵活

- 支持丰富接口：1000M 以太网口、USB3.0/USB2.0、HDMI、RS-485、RS-232；
- 支持 -20℃ 至 +60℃ 宽温工作环境；
- 支持 IP40 防护等级；
- 支持 Wi-Fi/BT 无线互联；
- 支持扩展硬盘存储(M.2 SSD);
- 可选配支持 4G 模块。

#### 易用工具链，开发灵活

- 一站式深度学习开发工具包 Sophon SDK;
- 支持 Caffe/DarkNet/TensorFlow/PyTorch/MXNet/ONNX/PaddlePaddle 等主流深度学习框架；
- 支持分类、检测主流网络模型，支持自定义算子开发；
- 支持 Docker 容器化，算法应用快速部署。

### 4.2 设备操作

#### 4.2.1 下电

SE7 支持以下两种下电方式。

**方式一 电源开关按键下电**

- 步骤 1 长按前面板上的电源开关（如下图虚线框所示）按键 3 秒钟。
- 步骤 2 观察电源指示灯（绿色）熄灭，表示 SE7 下电成功。
- 结束

**方式二 电源线缆下电**

- 步骤 1 拔掉电源线缆，将 SE7 下电。
- 步骤 2 观察电源指示灯（绿色）熄灭，表示 SE7 下电成功。
- 结束

#### 4.2.2 上电

SE7 支持以下两种上电方式。

**方式一 电源开关按键上电**

- 步骤 1 长按前面板上的电源开关按键 2 秒钟，将 SE7 上电。
- 步骤 2 观察电源指示灯呈现绿色常亮状态，确保 SE7 上电成功。
- 结束

**方式二 电源线缆上电**

- 步骤 1 连接电源线缆，将 SE7 上电。
- 步骤 2 观察电源指示灯呈现绿色常亮状态，确保 SE7 上电成功。
- 结束

### 4.4 选件安装

#### 4.4.1 硬盘安装（可选）

**说明**

- 硬盘功能为可选功能，支持 SSD 硬盘。
- 硬盘支持容量不超过 2TB，为非标配配件，需要用户根据需求自行采购。

**操作步骤**

- 步骤 1 将 SE7 下电。
- 步骤 2 拆卸 SE7 的电源线缆、Wi-Fi/BT 天线、4G 天线。
- 步骤 3 准备工具：PH0、PH1 螺丝刀，如图所示。
- 步骤 3 将 SE7 放置于防静电台，用 PH1 螺丝刀拆掉后盖 4 颗螺丝。
- 步骤 4 用 PH1 螺丝刀，拆掉底盖 6 颗螺丝。
- 注：这里一共有6颗螺丝，靠近后盖的2颗是短的，另外4颗是长的。
- 步骤 5 用手轻轻扣开顶盖，主板有风扇连接线连接到顶盖，小心操作、不要扯断。之后，拔掉主板上的风扇连接线，完全取下顶盖。
- 步骤 6 使用 PH2 螺丝刀，拆掉底板四个角上的四颗螺丝，取出底板。
- 步骤 7 翻转底板，使用PH1螺丝刀，拆掉底板背面固定固态硬盘的螺丝。
- 步骤 7 更换新固态硬盘。
- 步骤 8 更新硬盘后装机步骤与上述步骤相反操作。
- 结束。

#### 4.4.2 V2 版本硬盘安装方式

SE7 32-EA4-23 V2 版本在底板上开了维护窗，可直接通过此维护窗更换硬盘。

### 4.5 设备安装

#### 4.5.1 桌面安装

**⚠️ 说明**

- 保证桌面平稳且通风散热环境良好。
- 严禁遮挡、覆盖 SE7 面板出风口或壳体表面。

**操作步骤**

- 步骤 1 将 SE7 静置于桌面。
- 步骤 2 连接外接线缆，参见 4.6 外接线缆。
- 步骤 3 将 SE7 上电，参见 4.2.2 上电。
- 结束

### 4.6 外接线缆

#### 4.6.1 以太网线

**准备工作**

- 准备工具：防静电手环或腕带。
- 准备辅材：超五类网线，建议使用带屏蔽双绞线。

**操作步骤**

- 步骤 1 将防静电手环或腕带佩戴。
- 步骤 2 将网线一端连接至 SE7 设备网口上，将另一端连接至对端设备网口上。
- 结束

#### 4.6.2 HDMI 线缆

**说明**

- 若用户定制开发应用软件，需要自行开发 HDMI 的软件应用程序。

**操作步骤**

- 步骤 1 将防静电手环或腕带佩戴。
- 步骤 2 将 HDMI 高清视频线缆一端插入 SE7 后面板的 HDMI 接口，另一端插入带有 HDMI 接口的对端设备显示器。
- 结束

#### 4.6.3 Wi-Fi/BT 天线

**说明**

- 天线为标配配件，产品包装内包含 3 根棒状天线。

**操作步骤**

- 步骤 1 将防静电手环或腕带佩戴。
- 步骤 2 将两根 Wi-Fi 天线紧固在 SE7 前面板标有 WIFI 丝印的 SMA 接插件上。
- 步骤 3 将一根 BT 天线紧固在 SE7 前面板标有 BT 丝印的 SMA 接插件上。
- 结束

#### 4.6.4 4G 天线（可选）

**说明**

- 无线功能为可选功能，以单独产品系列支持，详情请联系我司销售人员。
- 4G 天线为非标配配件，对于支持无线功能产品系列，标配 4G 天线供货。
- 4G 天线为垂直极化方式，建议设备桌放时垂直放置，挂墙时与设备平行放置。

**操作步骤**

- 步骤 1 将防静电手环或腕带佩戴。
- 步骤 2 将 4G 天线紧固在 SE7 前面板标有 4G 丝印的 SMA 接插件上。
- 步骤 3 将 4G 天线分别弯折至垂直（设备桌放）或与设备平行（设备挂墙）。
- 结束

## 5 系统配置

### 5.1 系统概述

- SE7 预装 Ubuntu 20.04，用户可以通过 SSH 登录或者串口登录（参考 6.5 章节），也可以通过 LiteOS 登陆。
- SE7 提供的 2 个网口，其中 WAN 口(eth0)：默认为 DHCP 动态获取 IP 由连接的路由分配。LAN 口(eth1)：默认为静态 IP，IP 地址为 192.168.150.1。
- SE7 默认登陆用户名/密码为 linaro/linaro 或 admin/admin。

### 5.2 系统登录

#### 5.2.1 SSH 登录

**说明**

首次登陆一般通过 LAN 口 IP 登陆，查看 WAN 口 IP 后可通过 WAN 口 IP 登陆。

**操作步骤**

- 步骤 1 准备两根网线，分别插入盒子 LAN 口和 WAN 口，其中 LAN 口网线接入电脑，WAN 口网线接入局域网内的交换机或者路由器。
- 步骤 2 配置电脑 IP 为 192.168.150.2，子网掩码为 255.255.255.0。
- 步骤 3 打开命令提示符，使用 ssh 登陆：ssh linaro@192.168.150.1，输入密码：linaro。
- 步骤 4 输入 ifconfig 命令查看 WAN 口 IP(eth0)，后续可以直接通过 WAN 口 IP 登陆。
- 结束

#### 5.2.2 Web 登录

SE7 Ubuntu 版本预装 LiteOS 后台管理系统，建议用户通过谷歌 Chrome 浏览器登录 LiteOS 系统，并进行系统配置。具体操作可参考《SOPHLITEOS 用户使用手册》。

#### 5.3.2 WAN IP 修改

**说明**

WAN IP 地址修改有多种方式，即 LiteOS 系统 web 页面修改、HDMI 界面修改；此外，您也可以通过串口或 SSH 登录到系统终端后，使用操作系统级别的通用命令进行修改。

- LAN IP 为 LiteOS 系统强制的静态 IP, LiteOS 系统修改 IP 的方式无法修改 LAN 口 IP。
- HDMI 界面虽然可以修改 LAN 口 IP，但不建议修改 LAN 口的 IP。
- WAN IP 和 LAN IP 不能处于同一网段，必要时可 SSH 登录到系统终端并使用操作系统通用命令修改 LAN 口 IP，但建议 LAN IP 保持设置为静态 IP，以方便运维管理。
- 本章节主要说明 HDMI 界面修改方式。

**操作步骤**

- 步骤 1 使用 HDMI 连接线连接 SE7 和显示器。

### 6.3 U 盘/TF 卡挂载

#### 说明

SE7 支持 U 盘和 TF 卡。

- 文件系统支持 FAT、FAT32、EXT2/3/4、NTFS。SE7 微服务器不支持自动挂载，所以需要手工进行 mount 挂载。完成数据写入后，请及时使用 sync 或 umount 操作，关机时请使用 sudo poweroff 命令，避免暴力下电关机，以免数据丢失。

#### 操作步骤

**U 盘挂载**

- 步骤1 插入U盘，使用使用 dmesg 可知道 U 盘对应的 sd 设备。

```
[15460.953423] [5] sdb: sdb1
```

- 步骤2 挂载U盘：

```bash
# 创建挂载目录
mkdir disk
# 挂载
sudo mount /dev/sdb1 disk
# 查看 U 盘内的文件
ls disk
```

**TF 卡挂载**

- 步骤1 插入TF卡，使用 dmesg 可知道 TF 卡对应的 mmcblk 设备
- 步骤2 挂载TF卡：

```bash
# 创建挂载目录
mkdir media
# 挂载
sudo mount /dev/mmcblk1p1 media
# 查看 TF 卡内的文件
ls media
```

### 6.4 硬盘挂载

#### 说明

- 硬盘功能为可选功能，支持 M.2 SATA3.0 SSD (2242)硬盘。
- 硬盘支持容量不超过 2TB，为非标配配件，需要用户根据需求自行采购。
- 使用前请先对硬盘进行格式化及分区操作，格式化文件系统类型请选择 "FAT32"、"EXT4"。

#### 操作步骤

- 步骤1 将防静电手环或腕带佩戴
- 步骤2 对硬盘进行格式化操作，并创建分区
- 步骤3 对格式化并分区好后的硬盘进行安装，参见 4.4.1 硬盘安装（可选）
- 步骤4 以 SSH 方式登录设备，可查看到 "/dev/sda1"、"/dev/sda2" 等盘符
- 步骤5 创建文件夹，键入命令 "mkdir /media/hd-sda1"
- 步骤6 挂载硬盘，键入命令 "mount /dev/sda1 /media/hd-sda1"

### 6.6 RS485/RS232

#### 6.6.1 RS485

#### 说明

- SE7 微服务器有一个 RS485 接口，设备名称为 /dev/ttyS2，支持全双工，默认波特率为 9600。

#### 操作步骤

具体测试步骤如下：

- 步骤1 连接硬件。将开发板 RS485 的A、B、GND 引脚分别和 PC 机串口适配器 (USB 转 485 转串口模块) 的 A、B、GND 引脚相连。
- 步骤2 打开 PC 机串口终端。在终端打开 kermit，并设置波特率：

```bash
$ sudo kermit  
C-Kermit> set line /dev/ttyUSB0  
C-Kermit> set speed 9600  
C-Kermit> set flow-control none  
C-Kermit> connect
```

其中 /dev/ttyUSB0 为 USB 转串口适配器的设备文件，注意要看实际 PC 机识别的为准。

- 步骤3 发送数据。RS232 的设备文件为 /dev/ttyS1 。在设备上运行下列命令：

```bash
sudo -s
stty -F /dev/ttyS2 9600 -echo
echo linaro RS485 test... > /dev/ttyS2C-Kermit
```

PC 机中的串口终端即可接收到字符串 "linaro RS485 test…"

- 步骤4 接收数据。首先在设备上运行下列命令：

```bash
sudo -s
cat /dev/ttyS2
```

然后在 PC 机的串口终端输入字符串 "linaro RS485 test…"，设备端即可见到相同的字符串。

#### 6.6.2 RS232

#### 说明

SE7 微服务器有一个 RS232 接口，设备名称为 /dev/ttyS1，支持全双工，默认波特率为 115200。

- 需要注意的是 RS232 可能会用作登录使用，因此用户若要使用，应当输入以下命令将其恢复成普通通讯串口功能：

```bash
sudo systemctl disable --now serial-getty@ttyS1.service
```

- 测试方法与 RS485 的步骤是类似的，只需要注意设备名称与波特率即可。

### 6.7 Wi-Fi 连接

#### 说明

SE7 微服务器默认支持 Wi-Fi 无线互联。

在使用 Wi-Fi 功能前，需要正确安排天线。请参考 4.6.3 章节。

- 官网的刷机包中没有内置 WIFI 芯片驱动，请参考 https://developer.sophgo.com/thread/578.html 此网页安装 WIFI 驱动。

#### 操作步骤

Wi-Fi 连接步骤如下：

- 步骤1 使能Wi-Fi：

```bash
nmcli radio wifi on
```

关闭Wi-Fi命令：

```bash
nmcli radio wifi off
```

- 步骤2 查看 Wi-Fi 是否使能成功：

```bash
# 打印出 enabled 表示成功
nmcli radio wifi
```

- 步骤3 查看 Wi-Fi接入点：

```bash
nmcli dev wifi list
```

- 步骤4 连接到 Wi-Fi接入点：

```bash
sudo nmcli device wifi connect zouxftest1 password 12345678 name test
```

其中 zouxftest1 为 WIFI 接入点的名称，12345678 则是密码。

连接成功日志如下：

```
Device 'wlan0' successfully activated with 'fd6c634b-f517-4ae9-a8e9-292a9c19d25c'.
```

#### Wi-Fi 热点操作步骤如下：

- 步骤1 使用 nmcli 命令可以创建无线 AP 热点：

```bash
sudo nmcli device wifi hotspot ifname wlan0 con-name my-hostapt ssid zouxftest7 band bg password 12345678 channel 5
```

说明如下：

- con-name：连接名称，这里定义为 my-hostapt
- ssid：创建的 AP 热点的名称，这里定义为 zouxftest7
- band：WIFI 的协议标准，这里选择 bg
- password：创建的 AP 热点的密码，这里定义为 12345678
- channel：创建的 AP 热点的通过，这里定义为 5

- 步骤2 打开/关闭 Wi-Fi热点：

```bash
sudo nmcli connection up[down] my-hostapt
```
